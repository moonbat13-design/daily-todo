# ✅ Daily Todo

카테고리별 할 일 관리 + 마감일 D-day + 키워드 자동 분류 기능을 갖춘 바닐라 웹 앱입니다.

## 미리보기

| 전체 목록 | 자동 분류 힌트 | 모바일 |
|-----------|---------------|--------|
| 진행률 바 · 카테고리 카드 | 입력 시 실시간 카테고리 감지 | 반응형 레이아웃 |

## 기능

- **할 일 CRUD** — 추가 / 완료 체크 / 수정(더블클릭 또는 ✏️) / 삭제
- **카테고리** — 💼 업무 · 🏠 개인 · 📚 공부
- **진행률 바** — 전체 및 카테고리별 실시간 업데이트
- **마감일 & D-day** — 날짜 선택 시 `D-Day` / `D-3` / `D+2` 배지 표시
- **키워드 자동 분류** — 입력 내용을 분석해 카테고리 자동 선택 및 힌트 표시
- **카테고리 필터** — 탭 클릭으로 원하는 카테고리만 보기
- **완료 항목 일괄 삭제**
- **데이터 영속성** — `localStorage`에 자동 저장

## 기술 스택

- HTML / CSS / JavaScript — 빌드 도구 없음, 프레임워크 없음
- 외부 라이브러리 없음 (순수 바닐라)

## 실행 방법

```bash
# 저장소 클론
git clone https://github.com/moonbat13-design/daily-todo.git
cd daily-todo

# index.html을 브라우저에서 열기 (서버 불필요)
start index.html   # Windows
open index.html    # macOS
```

## 파일 구조

```
daily-todo/
├── index.html   # HTML 구조
├── style.css    # 디자인 시스템 (CSS Variables, 반응형)
└── script.js    # 상태 관리, CRUD, D-day, 자동 분류 로직
```

## D-day 배지 색상

| 배지 | 조건 | 색상 |
|------|------|------|
| `D-Day` | 오늘 마감 | 노란색 |
| `D-1` ~ `D-3` | 3일 이내 | 빨간색 |
| `D-7` 이상 | 여유 있음 | 초록색 |
| `D+N` | 마감 초과 | 진한 빨간색 |

## 자동 분류 키워드 예시

| 카테고리 | 키워드 예시 |
|----------|------------|
| 💼 업무 | 회의, 미팅, 보고서, 발표, 기획, 마감, 클라이언트 |
| 🏠 개인 | 헬스, 운동, 병원, 친구, 쇼핑, 여행, 청소 |
| 📚 공부 | 공부, 과제, 시험, 독서, 코딩, 강의, 자격증 |
